Privacy Policy

Last Updated: May 11, 2026

Developer Response Protocol

Supported search pages update their underlying DOM structure and CSS selectors frequently. CleanSearch Pro is an actively maintained tool. The one-time $5 premium upgrade supports ongoing compatibility updates, bug fixes, and blocking-rule maintenance.

Browsing Data Stays Local

No search queries, browser history, page contents, search result text, or document metadata are sent to CleanSearch Pro servers. The extension processes supported search result pages locally in your browser and hides matching AI Overview, Gemini, and sponsored-result elements on the page.

License Validation Data

To run trials, restore purchases, and validate premium access, CleanSearch Pro sends the minimum account-status data needed by the license server: a random install ID generated by the extension, the extension ID and version, activation status, and refresh-token or checkout metadata. The server stores sensitive identifiers as hashes where practical, including install IDs, legacy license keys, refresh tokens, and purchase email addresses when provided by a payment processor.

Payments

New premium purchases are handled through Stripe Checkout. CleanSearch Pro does not see, process, or store your card number, card security code, bank details, or other payment credentials. The server may store Stripe checkout, payment, and customer identifiers so it can activate purchases, restore access, and revoke access after refunds, disputes, or chargebacks. Legacy Gumroad license keys remain supported for customers who purchased before the Stripe migration, but new extension checkout uses Stripe.

No Selling or Analytics

CleanSearch Pro does not sell personal information, run behavioral advertising, or collect analytics about your searches. Support emails you send voluntarily are used only to answer your request and troubleshoot your account or extension install.

Host Permissions

The extension requires permission to google.com properties so it can identify and hide supported AI Overview, Gemini, and sponsored-result elements. This permission is used for on-page blocking only. CleanSearch Pro does not track user behavior or modify search rankings.

Retention & Requests

License and payment records are retained as long as needed to provide premium access, support restorations, handle refunds or disputes, prevent abuse, and meet basic accounting obligations.
